🧭 دليل شامل للوحة الإدارة `admin.html`

كلمة المرور: 11121999

🏆 1. إدارة الجوائز (الجزء الأزرق العلوي)

  • 🔍 جلب الجوائز الحالية: يجلب قيمة الجائزة المحفوظة للمستوى من Firebase ويملأ الحقل
  • 💾 حفظ جائزة المستوى: يحفظ القيمة في Firebase للمستوى المختار
  • 🔄 إعادة الافتراضي: يحذف الجائزة ويعود للقيمة الافتراضية (10,25,50... جنيه)

📝 2. محرر الأسئلة (الجزء الأبيض)

الخطوات:

  1. اختر المستوى من القائمة المنسدلة
  2. 🔍 جلب الأسئلة الحالية: يجلب جميع أسئلة المستوى ويعرضها في الـ textarea
  3. عدّل الأسئلة في الـ textarea (يدعم التنسيقين الجديدين!)
  4. 🧪 اختبار JSON: يفحص الأسئلة قبل الحفظ ويعرض المعاينة + الأخطاء
  5. 💾 حفظ التعديلات: يحفظ في Firebase بالصيغة الصحيحة
  6. 🗑️ مسح أسئلة المستوى: يمسح **كل** أسئلة المستوى نهائياً

✅ التنسيقين المدعومين (الجديد):

{ question: "سؤالك هنا؟", options: ["الخيار الأول", "الخيار الثاني", "الثالث"], answer: "الخيار الأول" // نص مطابق أو رقم (0,1,2...) }
{"question": "نفس السؤال", "options": ["الخيار الأول", "الخيار الثاني"], "answer": 0}

ملاحظة: يحوّل تلقائياً {question: "..."} → {"question": "..."} للتوافق مع JSON

🎯 مثال كامل للاستخدام:

  1. اختر "المستوى 1"
  2. اضغط "🔍 جلب الأسئلة الحالية"
  3. الصق في textarea:
    [ { question: "ما عاصمة مصر؟", options: ["القاهرة", "الإسكندرية"], answer: "القاهرة" } ]
  4. اضغط "🧪 اختبار JSON" → ✅ يظهر معاينة خضراء
  5. اضغط "💾 حفظ" → ✅ "تم الحفظ!"
