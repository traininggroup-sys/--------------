# كيف تضيفي أسئلة لكل مستوى بدون مشاكل - دليل كامل

## التشخيص: ليه مش بتشتغل الأسئلة؟
1. **الأسئلة في Firebase** مش محلية في الكود.
2. **المشكلة الشائعة**: JSON غلط في admin.html (answer مش index أو نص مطابق).
3. **التخطي التلقائي**: لو سؤال ناقص، اللعبة بتخطيه وتروح next.

## خطوات إضافة أسئلة (مضمونة تعمل):

### 1. افتحي admin.html
```
- اضغطي ⚙️ من index.html أو افتحي admin.html مباشرة
- كلمة مرور: 11121999
```

### 2. اختاري المستوى + جيبي الأسئلة الحالية
```
اختاري المستوى 1 ← 🔍 'جلب الأسئلة الحالية'
لو فاضي: هيظهر []
```

### 3. **الصيغة الصحيحة 100% للـ JSON** (انسخي + غيري):
```
[
  {
    \"question\": \"عاصمة مصر؟\",
    \"options\": [\"القاهرة\", \"الإسكندرية\", \"الجيزة\"],
    \"answer\": 0
  },
  {
    \"question\": \"2 + 2؟\",
    \"options\": [\"3\", \"4\", \"5\"],
    \"answer\": 1
  }
]
```
**مهم جداً:**
- `answer`: رقم الفهرس (0 للأول, 1 للتاني...)
- OR نص مطابق بالظبط: `"answer": "القاهرة"`
- options 2+ خيارات min
- question مش فاضي

### 4. الصقي في textarea + اختبري
```
اضغطي 🧪 'اختبار JSON'
لو أخضر: ✅ تمام
لو أحمر: عدلي الخطأ المكتوب
```

### 5. احفظي
```
💾 'حفظ التعديلات البرمجية'
→ Toast أخضر = نجح
```

### 6. اختبري في اللعبة
```
1. index.html → تسجيل دخول
2. اختاري المستوى 1 → ابدأ
3. افتحي Console (F12)
4. شوفي logs: Loaded X questions
5. اللعب الطبيعي
```

## لو لس مش شغال:
1. **Console logs**: شو اللي مكتوب؟ (copy-paste هنا)
2. **Firebase Console**: https://console.firebase.google.com → Realtime DB → questions/1
3. **مسح وإعادة**: 🗑️ Clear → أضيفي واحد بس أول
